## Elasticsearch - Logstash - Kibana 

Make sure you have docker

### Building the ELK
    - clone
    - cd docker-elk

### Build Containers

    docker-compose build 

### Running Containers
    docker-compose up
    
### Test
    $ curl http://localhost:8080/user/{userid}
   
Kibana UI: [http://localhost:5601](http://localhost:5601)
